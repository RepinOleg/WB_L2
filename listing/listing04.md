Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет числа от 0 до 9 после чего произойдет deadlock
Цикл который читает из канала работает до тех пор пока канал не будет закрыт.
После того как цикл внутри горутины запишет все числа и горутина завершит свою работу, цикл на чтение будет продолжать ожидать значений из канала
Из-за этого возникает deadlock
Исправить эту ошибку можно закрыв канал, после записи всех нужных значений

```
